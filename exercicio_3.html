Exercício 3

Faça uma aplicação que verifique se um valor é perfeito ou não. Um valor é dito perfeito quando ele é igual a soma dos seus divisores, exceto ele próprio. (Ex: 6 é perfeito, 6 = 1 + 2 + 3, que são seus divisores).

Código feito em linguagem C
Valor Perfeito

#include <stdio.h>
#include <stdlib.h>
#include <locale.h>//Biblioteca de caracteres especiais

 

int main()
{
setlocale(LC_ALL, "");//Biblioteca de caracteres especiais

int n, soma= 0, i;

printf("Digite um número maior que 0: ");
scanf("%d", &n);

for(i = 1; i < n; i++)
{
if(n % i == 0)
{
soma += i;
}
}

if (soma == n)
{
printf("\n\nO número %d é perfeito!!\n\n", n);
}
else
{
printf("\n\nO número %d não é perfeito!!\n\n", n);
}

return (0);

}
